# Kavi VS — Portfolio
 
Live: [kavi-vs-portfolio.netlify.app](https://kavi-vs-portfolio.netlify.app)
 
Personal portfolio website for Kavi VS, a frontend developer based in Chennai, Tamil Nadu. Built as a static, single-page site showcasing skills, projects, experience, and contact information.
 
## Tech Stack
 
- HTML5
- CSS3 (custom styles, no framework)
- Vanilla JavaScript
- Google Fonts (DM Sans)
## Project Structure
 
```
portfolio/
├── portfolio.html     # Main portfolio page
├── project1.html       # Case study page (SVARA Jewels)
├── style.css           # Global styles
├── resume.pdf          # Downloadable CV
├── PROJECT1.mp4         # Project demo video
└── images/              # Icons, profile photo, project visuals
```
 
## Sections
 
- **Hero** — Introduction, availability status, and key stats
- **Skills** — Tech stack: React, JavaScript, Tailwind CSS, Git/GitHub, ShadCN UI, WordPress, Bootstrap, and more
- **Projects** — Featured and mini project showcases:
  - Expense Tracker (React 19, Tailwind v4, Recharts, Vite)
  - Integrated College Web Portal (Flask, Python, Dialogflow, JavaScript)
  - E-Learning Landing Page (Tailwind CSS, JavaScript, Bootstrap)
  - SVARA Jewels e-commerce site (HTML/CSS, JavaScript, Bootstrap)
- **Experience & Education** — Web Design Internship (Phoenix Softech), MCA and B.Sc. Mathematics (SFR College for Women)
- **Contact** — Links to connect for remote/full-time frontend roles
## Running Locally
 
1. Clone the repository
```bash
   git clone https://github.com/VSkavi/portfolio.git
```
2. Open `portfolio.html` directly in a browser, or serve with a local server:
```bash
   npx serve .
```
