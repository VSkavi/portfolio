# Kavi VS — Portfolio

## Projects
 
### SVARA Jewels — E-commerce Website
A luxury jewellery shopping experience designed with an elegant UI, responsive product catalogue, and smooth checkout flow.
 
**Stack:** HTML · CSS · JavaScript · Bootstrap
 
---
 
## Tech Stack
 
- **HTML5** — semantic markup
- **CSS3** — custom properties, flexbox, grid, scroll-reveal animations
- **Vanilla JavaScript** — nav scroll effects, intersection observer
- **Google Fonts** — DM Sans
- **Bootstrap** — grid and utility classes (project pages)

## Getting Started
 
No build step required — open `portfolio.html` directly in a browser or serve with any static file server:
 
```bash
# Python
python -m http.server 8000
 
# Node
npx serve .
```
